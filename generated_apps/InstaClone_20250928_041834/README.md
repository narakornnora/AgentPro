# InstaClone

แอปแชร์ภาพถ่ายและวิดีโอที่ให้ผู้ใช้สามารถโพสต์และติดตามเพื่อน ๆ ได้

## การติดตั้งและรัน

```bash
npm install
npx react-native run-android
npx react-native run-ios
```
