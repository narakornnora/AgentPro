# InstaClone

แอปพลิเคชันโซเชียลมีเดียที่คล้าย Instagram สำหรับแชร์รูปภาพและติดตามผู้ใช้

## การติดตั้งและรัน

```bash
npm install
npx react-native run-android
npx react-native run-ios
```
